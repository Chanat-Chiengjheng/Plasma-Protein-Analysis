# Analysis Tools

The two ready-to-run programs for this project. See the
[project README](../README.md) for what each one does and how to run it —
short version:

```
pip install -r ../requirements.txt
python "Image_Analysis.py"
python "CSV_Analysis.py"
```

- `Image_Analysis.py` — measures plasma protein aggregates from photographs.
- `CSV_Analysis.py` — compares oscilloscope waveform CSV recordings.

When these files are updated, just re-download this folder (or `git pull`
the repo) — the file names don't change, so nothing else about how you run
them does either.
